# Commit ændringer

1. Se hvilke filer der er ændret
```console
git status
````

2. ```add``` til staging
```console
git add [filnavn]
```
eller
```console
git add .
````
for at tilføje alt

3. ```commit``` ændringer
```console
git commit -m "[commit besked]"
```

# Push ændringer

1. Commit ændringer først
2. ```pull``` ændringer
```console
git pull origin main
```
3. ```push``` ændringer
```console
git push origin main
```

# Pull ændringer
1. Commit ændringer først
2. ```pull``` ændringer
```console
git pull origin main
```
